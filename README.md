# flowhi (フロー日)

Frontend: Vite.js+React
Backend:Express.js
DB:Mysql

## Install
You need to install npm and node (>22)
```
cd frontend
npm install 
```

## Start
### Start backend: 
```bash
cd backend 

npm run dev
```

### Start frontend:
```bash
cd frontend

npm run dev
```

### Start Tailwind watch:
```bash
cd frontend

npm start
```
